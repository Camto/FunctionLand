• Diagonals: hue(x + y)
  • The color spectrum: hue(x / w * 360)
  • Diamond: hue((Math.abs(x - w / 2) + Math.abs(y - h / 2)) / 2 / 0.875 + 170)
  • Sine wave: hue(Math.min(Math.abs(y - h / 2 - Math.round(Math.sin(x / 100 * Math.PI) * 35)) + 180, 240))
  • Red is distance in x, blue is distance in y: [Math.abs(x - w / 2), 0, Math.abs(y - h / 2)]
  • Circle: hue(Math.round(Math.sqrt(((x - w / 2)**2 + (y - h / 2)**2))) / 1.25 + 170)
  • Square: hue(Math.max(Math.abs(w / 2 - x), Math.abs(h / 2 - y)) + 170)
  • Beveled square: hue(Math.max(Math.max(Math.abs(w / 2 - x), Math.abs(h / 2 - y)), (Math.abs(x - w / 2) + Math.abs(y - h / 2)) / 2 / 0.875) + 170)
  • 8-pointed star: hue(Math.min(Math.max(Math.abs(w / 2 - x), Math.abs(h / 2 - y)) / 1.0625, (Math.abs(x - w / 2) + Math.abs(y - h / 2)) / 2 / 0.75) + 170)
  • Checkerboard pattern: Array(3).fill(0).map((_, i) => (Math.floor(x / 20) % 2 == Math.floor(y / 20) % 2) * 255 + hue(x + y)[i] / 1.5)
